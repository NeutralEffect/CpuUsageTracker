{\footnotesize\ttfamily int Watchdog\+Thread (\begin{DoxyParamCaption}\item[{void $\ast$}]{params }\end{DoxyParamCaption})}



Thread function for detecting unresponsive threads and terminating program. 

Thread will periodically check activity reports supplied by other threads and compare their timing against predefined maximum allowed unresponsiveness period. If one or more of these reports are delayed beyond allowed period, this thread will activate kill switch requesting all threads to immediately terminate.
