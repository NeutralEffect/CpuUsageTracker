Thread function for reading /proc/stat file and parsing it\textquotesingle{}s content into Proc\+Stat\+\_\+t structure. 

Thread will periodically read and parse /proc/stat file and write the result into circular buffer provided through params. Those writes will be only performed if provided mutex is available, as it will be locked for their entire duration.
